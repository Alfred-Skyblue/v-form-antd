# 异步属性

当我们使用下拉菜单，树形选择，复选框组，单选框组等控件时，可能会遇到从服务器端获取数据的情况，我们支持在 props 中传递一个函数返回 Promise，用来获取 options 数据。

注：为了避免每次打开表单都发送异步请求，我们还是建议您在外部获取到选项以后，再传递到 props 参数中，当然如何使用是您的选择。

提交
<template>
  <div>
    <v-form-create :formConfig="formConfig" :formData="formData" v-model="fApi" />
    <a-button @click="submit">提交</a-button>
  </div>
</template>
<script>
export default {
  name: 'Demo',
  data() {
    return {
      fApi: {},
      formData: {},
      formConfig: {
        formItems: [
          {
            type: 'select',
            label: '下拉选择',
            field: 'select_1',
            span: 24,
            props: {
              options: () => {
                return new Promise(resolve => {
                  setTimeout(() => {
                    resolve([
                      {
                        label: '选项1',
                        value: '1'
                      },
                      {
                        label: '选项2',
                        value: '2'
                      }
                    ])
                  }, 1000)
                })
              }
            }
          },
          {
            type: 'treeSelect',
            label: '树形选择',
            field: 'tree_select_2',
            span: 24,
            props: {
              treeData: () => {
                return new Promise(resolve => {
                  setTimeout(() => {
                    resolve([
                      {
                        label: '选项1',
                        value: '1',
                        children: [
                          {
                            label: '选项三',
                            value: '1-1'
                          }
                        ]
                      },
                      {
                        label: '选项2',
                        value: '2'
                      }
                    ])
                  }, 2000)
                })
              }
            }
          },
          {
            type: 'checkboxGroup',
            label: '复选框',
            field: 'checkbox_group_3',
            span: 24,
            props: {
              options: () => {
                return new Promise(resolve => {
                  setTimeout(() => {
                    resolve([
                      {
                        label: '选项1',
                        value: '1'
                      },
                      {
                        label: '选项2',
                        value: '2'
                      }
                    ])
                  }, 3000)
                })
              }
            }
          }
        ],
        config: {
          layout: 'horizontal',
          labelLayout: 'flex',
          labelWidth: 100,
          labelCol: {},
          wrapperCol: {}
        }
      }
    }
  },
  methods: {
    async submit() {
      const data = await this.fApi.submit()
      alert(JSON.stringify(data, null, '\t'))
    }
  }
}
</script>
